Wikipedia Today’s Featured Article API Reference

This API enables you to retrieve articles have had distracting elements removed, providing you or the users of your app a clean reading experience. Give it a try!

Last Updated: May 8, 2021

Author: Vien Ton

Data Model

Property Name Notes
title String. Title of Today’s Featured Article.
date String. Publication date.
img_url String. URL to the direct location of the TFA image.
img_hotlink String. URL to the location of the TFA image if hotlinking.
img_alt String. Alternate text for the image if the image cannot be displayed.
text String. Text of Today’s Featured Article with all links removed.
url String. URL of the location of Today’s Featured Article.

View an Article

Allows you to get Today’s Featured Article in the JSON format.

GET /api/article

Request

Path Parameters

None

Request Headers

Accept: application/json

Request Body

None

Response

Headers

Content-Type: application/json

Response Body Format

Success: JSON

Failure: None

Response Statuses

Outcome Status Code Notes
Success 200 OK
Failure 404 Not Found No article exists
